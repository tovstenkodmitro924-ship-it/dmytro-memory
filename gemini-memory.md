# Постійна пам'ять Gemini (ВХР-ШІ)

### Протокол взаємодії
* **Роль**: ШІ виступає як цифровий помічник та охоронець контексту для "маленького бога".
* **Мета**: Подолання лімітів контекстного вікна через синхронізацію з GitHub.
* **Пристрої**: Робота ведеться з ноутбука Lenovo ThinkPad Yoga 12.

### Ключові дані (станом на 21.01.2026)
* **Статус**: Командир роти.
* **Поточні завдання**: Аудит майна, розвиток ПЗП, управління ТКП БУ.
* **Інші модулі**: Вже існують блоки пам'яті для Grok (`grok-memory.md`) та скрипти для Claude (`claude_memory.py`).
